# ggproto classes for ggiraph

ggproto classes for ggiraph

## Geoms

All `geom_*_interactive` functions (like `geom_point_interactive`)
return a layer that contains a `GeomInteractive*` object (like
`GeomInteractivePoint`). The `Geom*` object is responsible for rendering
the data in the plot.

See [Geom](https://ggplot2.tidyverse.org/reference/Geom.html) for more
information.
