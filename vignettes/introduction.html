ggiraph

David Gohel

2015-08-02

Package ggiraph make ggplot graphics interactive using htmlwidgets and ReporteRs. Available options are:

  • tooltips
  • javascript click action
  • javascript double click action

For now, five different geom are existing:

  • geom_path_interactive
  • geom_rect_interactive
  • geom_point_interactive
  • geom_segment_interactive
  • geom_polygon_interactive

Demo

library(ggiraph)
## Loading required package: ggplot2
dataset = iris
dataset$tooltip = paste0( "Species <br/>", dataset$Species )
dataset$clickjs = paste0("alert('",dataset$Species, "');" )

# plots
gg_point_1 = ggplot(dataset, aes(x = Sepal.Length, y = Petal.Width, 
        color = Species, tooltips = tooltip, clicks=clickjs) ) + 
    geom_point_interactive()
ggiraph(fun=print, x = gg_point_1)
